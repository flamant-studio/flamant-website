A trip down
memory lane.

Before Trustpilot, before Kahoot!, before any of it — there was flamant.dk. A personal website built in a Copenhagen flat, by hand, in Dreamweaver. Here's what the internet looked like when I was figuring it out.

2004

Den kedelige mand

My first personal website. Written in the third person. Featured sections on hvem (who), hvad (what) and a page called "schtroumpf" that I warned visitors about in advance. Introduced myself as someone who enjoys interactive storytelling, card games and the occasional cognac. Table-based layout, Arial throughout, ISO-8859-1 encoding. A document of its time.

HTML 4.01 Table layouts Dreamweaver Danish
Apr 2004

flamant.dk v2 — with a comment system

A proper redesign with navigation buttons, a sliced-image header and a live comment system built in PHP from scratch. Sections: intro, snak (chat), cv, arkiv, skriv (write). The blog before blogs were called blogs. Uploaded 1 May 2004 — it says so right in the header.

HTML + PHP MySQL Image slicing Comment system
2005–07

flamant.dk v3 — the full stack era

The most ambitious version. A full PHP/MySQL CMS with login, admin panel, article management, reviews, quizzes, RPG character sheets and a film database with CRUD. Separate sub-sites for a novel project, a Blood Bowl league, a film review system and a Smurf fan page. Running simultaneously on a single domain. Held together with optimism and nested tables.

PHP + MySQL Admin panel Film database Login system Multi-site
2007

The last redesign before the career took over

Final version of the personal site before digital became a profession rather than a hobby. Still PHP, still MySQL, but the sections had matured: articles, reviews, good advice, curiosities. Copyright notice reads 2005. Soon after this, the personal site quietly retired and the day job began in earnest.
